# Self-hosted fonts

These WOFF2 files support the Postmarked "Greetings From" theme and are served locally from `/fonts/`.

Sources:
- Playfair Display: Google Fonts / Claus Eggers Sorensen
- Source Serif 4: Google Fonts / Adobe
- IBM Plex Mono: Google Fonts / IBM

Licensing:
- These families are distributed through Google Fonts under the SIL Open Font License 1.1. Keep this directory limited to redistributable font files and update this note if the font sources change.

Implementation note:
- The current checked-in files are Latin WOFF2 subsets, matching the current English-language content needs.
